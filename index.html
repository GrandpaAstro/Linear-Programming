Linear Programming is a technique that helps identify the maximum or minimum of a quantity.

Steps

1) Identify the main function
2) Figure out the constraints
3) Graph the constraints
4) Find the coordinates for each vertex
5) Substitute each vertex into the main function, to find the maximum or minimum value.
